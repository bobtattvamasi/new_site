# my-site skeleton

Минимальный Astro 4 + TypeScript каркас без демонстрационных страниц. Всё, что нужно для мгновенного запуска dev-сервера и дальнейшего наращивания фич на Supabase.

## Структура

```
├─ src/
│  ├─ components/    # Header/Footer и будущие повторно используемые блоки
│  ├─ layouts/       # Общие обёртки, сейчас BaseLayout
│  ├─ pages/         # Маршруты (index.astro — единственная страница)
│  └─ env.d.ts       # Типы env-переменных
├─ public/           # favicon, статика
├─ .env.example      # Заглушки для Supabase ключей
└─ astro.config.mjs
```

## Скрипты

| Команда       | Описание                                    |
| ------------- | ------------------------------------------- |
| `pnpm install` | Ставит зависимости (Node 20 + pnpm 8/9/10) |
| `pnpm dev`     | Dev-сервер на http://localhost:4321        |
| `pnpm build`   | Сборка статики в `dist/`                   |
| `pnpm preview` | Просмотр уже собранного билда              |

## Дальше

- Конфигурация Supabase: копируй `.env.example` → `.env.local` и заполни ключи.
- Новые страницы — только в `src/pages`, шарим компоненты через `src/components`.
- Перед пушем: `pnpm build` (быстрая проверка) и линтер/тесты, когда появятся.
